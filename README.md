# USaP - Email Phishing Aid

Usable Security and Privacy Project - Fall 2025

Uses AI to give information on why an email was flagged as a potential phishing threat, goal is to increase user awareness on how phishing attacks can be detected.

This project is a ThunderBird email client extension

1. Install thunderbird - https://www.thunderbird.net/en-US/download/
2. Setting -> Add-ons and Themes -> Settings -> Debug Add-ons
3. Select Load Temporary Add-on and select manifest.json
